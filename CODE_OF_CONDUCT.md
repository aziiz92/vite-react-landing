# Code of Conduct

This project is a community space. By participating, you agree to keep it respectful and constructive.

## Be respectful

- Assume good intent and be kind in feedback.
- No harassment, hate speech, or discrimination.
- No doxxing or sharing private information.

## Be constructive

- Keep discussions on-topic and focused on improving the project.
- Prefer small, reviewable changes over large rewrites.

## Scope

Maintainers may remove or edit content that violates this Code of Conduct, and may block accounts for repeated or severe violations.

## Reporting

If something goes wrong, open an issue or contact the maintainers through the security process in `SECURITY.md`.
